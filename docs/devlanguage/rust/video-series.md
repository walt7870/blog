## 推荐观看路径

初学者可以按三轮推进：

1. 先看 01-03，理解 Rust 为什么强调所有权和借用。
2. 再看 04-14，把类型系统、错误处理、并发、异步、测试和工程工具串起来。
3. 最后看 19-26，按视频做一个 CLI 和一个小 Web 服务，把知识落到可运行项目里。

已经有其他语言经验的开发者，可以先看 01、02、03、05、07、09、10、14、17，再进入 19-26 的实操补齐。

## 学习闭环

只看概念还不够，Rust 最关键的是把编译器反馈变成日常开发节奏：

- 用 `cargo check` 高频检查类型和借用问题。
- 用 `cargo test` 固化核心行为。
- 用 `cargo clippy --all-targets --all-features` 提前发现质量问题。
- 用 `README`、文档测试和示例代码保持项目可交接。
- 用小项目验证依赖、配置、日志、错误处理和发布流程。

视频中的每个要点都可以在配套文章里找到可复制命令、排错路径或项目落地建议。
